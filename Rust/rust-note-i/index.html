发表于 2020-06-11 分类于 Rust 阅读次数： Valine：

Rust 的字符串本质上是 u8 序列，但是很多时候从 u8 序列转换回字符串是有风险的，毕竟并不是所有的 u8 序列都是有效的 字符串。
所以原则上来说，能直接使用 char 对字符串处理是最好的。
